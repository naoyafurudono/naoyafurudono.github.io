# おまけ

## ssh-keygenを便利に

```fish
function _ssh-keygen
  set name $argv[1]
  mkdir -p "$HOME/.ssh/$name"
  ssh-keygen -t ed25519 -f "$HOME/.ssh/$name/id_ed25519"
end
```

```bash
#!/bin/bash

name=$1
mkdir -p "$HOME/.ssh/$name"
ssh-keygen -t ed25519 -f "$HOME/.ssh/$name/id_ed25519"
```

## .ssh/configの例

```config
Host github.com
  User git
  IdentityFile /Users/naoyafurudono/.ssh/git/id_ed25519
```
