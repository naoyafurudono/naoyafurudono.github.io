# `envsubst`を知った

`envsubst`を使うと標準入力の環境変数の参照っぽいところに環境変数の値を埋め込んだテキストを吐ける。
readmeにコマンドのヘルプメッセージを載せたいときに便利。

# `git ls-files`

gitで管理しているファイルのパスが得られる。
git grepみたいに管理対象だけに興味があるときに便利そう。
fzfとの相性が良さげ。
